# bevlioon

Bevlioon is a simple and elegant movie exploration app.

## Getting Started

Bevlioon is a modern, lightweight movie discovery app built for film lovers. 

Explore the latest popular movies using The Movie Database (TMDB) API, enjoy smooth infinite scrolling,
and dive into rich film details including posters, ratings, and overviews. 

Save your favorite films locally on your device and personalize each one with a custom alias. 
With offline persistence, intuitive navigation, and a clean UI powered by Flutter & BLoC architecture, 
Bevlioon offers a fast and enjoyable way to manage your movie world.
